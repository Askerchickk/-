Этот скрипт Python использует библиотеку EasyOCR для обнаружения и распознавания российских автомобильных номеров на изображениях. Скрипт анализирует изображение, идентифицирует текст на номерных знаках с поддержкой русского и английского языков, и отображает распознанный текст прямо на изображении с помощью OpenCV. Кроме того, скрипт добавляет метку времени на изображение, указывая текущую дату и время обработки.

Основные функции:
Распознавание текста на автомобильных номерах: Использует EasyOCR для идентификации и распознавания текста.
Аннотация изображений: Нарисовывает рамки вокруг распознанных номерных знаков и добавляет распознанный текст непосредственно на изображение.
Метка времени: Вставляет текущую дату и время на обработанное изображение.
Сохранение результатов: Сохраняет аннотированное изображение и записывает распознанные номерные знаки в текстовый файл с указанием времени обнаружения.
Использование:
Установите необходимые библиотеки: easyocr, opencv-python.
Запустите скрипт, указав путь к изображению с автомобильными номерами.
Просмотрите результаты в созданном текстовом файле и аннотированном изображении.
Этот скрипт полезен для проектов, связанных с автоматизацией мониторинга транспортных средств, систем безопасности и управления доступом.
